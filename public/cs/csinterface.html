Mi az az interfész?

Az interfész egy olyan teljesen absztrakt osztály, amely csak metódusok deklarációit tartalmazza. Az interfészek lehetővé teszik a többszörös öröklődést, mivel egy osztály több interfészt is implementálhat. Erre a : szimbólumot használjuk. Az interfészben lévő metódusokat az implementáló osztálynak kell megvalósítania.

Megvalósítandó metódusok a Auto osztály számára:

  • void indulas(); - Elindítja a járművet
  • void megallas(); - Megállítja a járművet
  • int getSebesseg(); - Visszaadja az aktuális sebességet

Interfész deklaráció

public interface Jarmu { void indulas(); void megallas(); int getSebesseg(); }

Implementáló osztály

public class Auto : Jarmu { @Override public void indulas() { sebesseg = 10; } @Override public void megallas() { sebesseg = 0; } @Override public int getSebesseg() { return sebesseg; } }
